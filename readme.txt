
Various productivity tools used while at IEI for dumping correspondence

CountAccounts.pl ............ Traverses dirs, counts all the dumped acccounts
DumpTXCAFiles.pl ............ Dumps all TX correspondence from CorrespondenceArchive
DumpTXMPAFiles.pl ........... Dumps all TX correspondence from MPA
PatchCSVHeaders.pl .......... Prepends a header to each index.csv file
RemoveEmptyAccountDirs.pl ... Removes all account dirs with no Correspondence
